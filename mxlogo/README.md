# 傲游 PC 客户端新标签页网站 logo 图片

## 使用地址

- 国外 `userfile.maxthon.com/file/mx-pub/logo/...`
- 国内 `pub.mxfast.net/file/mx-pub/logo/...`

## 说明

- 本目录仅用于保存文件和管理版本，每次更新图片后，还需要使用 Cyberduck 客户端同步更新，才能在项目中使用
- 图片有两种格式，`rectangle` 目录下为长方形，`square` 目录下为正方形，新增时两种格式都要有

## 来源

- 原始图片在 `\static\v1.1\image\logo` 目录下，由于命名混乱，在 2025-12-23 重新整理后迁移到本目录，原目录不再使用
